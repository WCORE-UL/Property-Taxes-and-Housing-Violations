# Analysis Phase

## Map collection
This section is largely the same as the last few with one notable inclusion. The Tax/Violation selection allows the user to show
which areas  have high tax payments for their number of violations. In this case, those areas are the southwestern parts of MOCO and a
curious code up north. These results enforce the narrative that low density, high value housing has a lower rate of violations than 
high density high value housing. 

## Treemap
The bubble plot has been reworked into a treemap. This visualization makes it easier to see that the two classes of 038 and 042 make up ~75% of all of the violations in MOCO.

## Dotplot
An additional selection has been added to the dotplot, that being log 10. This drastically changes the pattern of the data from a 
heavily right skewed plot to a more normal but oddly striped distribution. Additionally, averages have been added. These averages show
just how much the data is skewed. 

## Big Map
This is a part of the project I have been working on for a while without a good idea on how to implement it. It visualizes ~600,000
points of geographic data, making for a hellish experience when attempting to mess with it. There is a wealth of information hidden in it however.
Clicking on one of the categories will display where those items are located, which highlights the discrepencies in accomodation and 
housing traits.
